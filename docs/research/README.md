# Research & Intelligence Sources

This directory contains research materials, intelligence source documentation, and reference notes used for the Geospatial HormuzWatch project.

## Intelligence Data Sources
- **AISStream**: Documentation on AIS message types and geofencing.
- **OpenSky**: REST API limits and state vector schemas.
- **GDELT**: GEO 2.0 API querying parameters for Geospatial and conflict events.
- **NASA FIRMS**: (Planned) Fire detection REST integrations.
